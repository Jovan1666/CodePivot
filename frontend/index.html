🎉 欢迎使用 CodePivot

一键切换 AI 编程工具配置，支持 Claude、Codex、Gemini、OpenCode

1

选择左侧厂家

点击 Claude、Codex、Gemini 或 OpenCode

2

添加配置

点击「添加配置」，填写 API 地址、密钥和模型名称

3

切换使用

选中配置后点击「切换使用」按钮，自动写入配置文件和环境变量

💡

配置不生效？

点击「部署环境变量」按钮，强制设置系统环境变量（适用于 CLI 工具读取不到配置的情况）

🎯 快速操作指南：

保存 保存当前配置
切换使用 应用配置到工具
部署环境变量 设置系统环境变量
删除 删除当前配置
